React 简介

React介绍：

用于构建用户界面的javascript库，主要用于构建Ui界面。

React特点：

1.声明式的设计；

2.高效，采用虚拟DOM来实现DOM渲染，最大限度的减少DOM的操作；

3.灵活，跟其他库可以灵活搭配使用。（不适合和vue配合两个库不要重复使用）

4.JSX，俗称：js里面写HTML,javaScript语法的扩展；

5.组件化，模块化，代码容易复用；

6.单向数据流。没有实现数据的双向绑定。数据 => 视图 => 事件=>数据;

创建react项目 （官方文档）

可以通过 CDN 获得 React 和 ReactDOM 的 UMD 版本。

生命周期

生命周期既是组件从实例化到渲染到最终销毁，整个过程就是生命周期，在这个生命周期中，我们有许多可调用的事件。也俗称为钩子函数

生命周期的三个状态：

Mounting: 将组件插入DOM中；

Updating : 将组件更新到DOM中；

Unmounting:将组件移除DOM中；

生命周期中的钩子函数（方法，事件）

CompontWillMount： 组件将要渲染 ，AJAX

CompontDidMount：组件渲染完毕,添加动画前的类

compontWillReceiveProps: 组件将要接受props数据，查看接受的props的数据是什么

shouldComponentUpdate: 组件接收到新的state或者props，判断是否要更新。返回布尔值；

CompontWillUpdate: 组件将要更新

CompontDidUpdate: 组件已经更新

CompontWillUnmount: 组件将要卸载

class ComLife extends Component{
  constructor(props){
    super(props) //调用继承Component的构造函数
    this.state = {
      msg: 'helloword'

  componentWillMount(){
    console.log('componentWillMount将要渲染');
  }
  componentDidMount(){
    console.log(' componentDidMount组件渲染完毕');
  }
  componentWillReceiveProps(){
    console.log(' componentWillReceiveProps组件将要接受新的状态state和props');
  }
  shouldComponentUpdate(){
    //如果希望更新，就返回为真，不希望更新就返回false
    console.log('进行判断要不要更新');
    if(this.state.msg == 'helloword'){
      console.log(this.state.msg);
      return true
    }else{
      console.log(false);
      return false
    }
  }
  componentWillUpdate(){
    console.log('componentWillUpdate 组件将要更新');
  }
  componentDidUpdate(){
    console.log('componentDidUpdate 组件更新完毕');
  }
  componentWillUnmount(){
    console.log(' componentWillUnmount 组件移除');
  }
  render(){
    console.log('render渲染函数')

      <div>
        <h1>{this.state.msg}</h1>

      </div>
    )

解决React数据管理（状态管理），用于中大型，数据比较庞大，组件之间数据交互多的情况下使用，如果你不知道是否需要用Redux，那么就不要使用它！

解决组件的数据通信

解决数据和交互多的应用

Redux 只是一种状态管理的解决方案！

store： 数据仓库，保存数据的地方。

State:state是一个对象，数据仓库里的所有数据都放在一个state中。

Action:1个动作，触发数据改变的方法；

Store：Dispatch： 将动作触发成方法；

Reducer:是一个函数，通过获取动作，改变数据，生成一个新的状态（state），从而改变页面

概念：

Provider组件： 自动的将store里的state和组件进行关联；

connect方法：将组件和数据（方法）进行连接
